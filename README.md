## General info
Browser based app for trivia game

Questions are sourced from The Trivia API (https://the-trivia-api.com/)

## Technologies
Project is created with:
* React
* Typescript
* Vite
	

### Initialize project
To run this project, install it locally using npm:

```
$ cd ../questionMaster
$ npm install
$ npm run preview
```
